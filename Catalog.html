Records Catalog

Browse and filter all searchable datasets in the JewishGen collection. Every entry links to a full description page and directly into the records. Use the filters below to find datasets by record type, region, or time period.

💡 Search Tips and Tricks
Tips for Using the Records Catalog:
Filter by Record Type: Narrow results to births, marriages, deaths, censuses, and more.
Example: Select "Census / Revision Lists" to find Czarist tax records
Filter by Region: Choose a country, then refine by historical sub-region or gubernia.
Example: Ukraine → Pale of Settlement → Kiev Gubernia
Date Range: Find records that overlap with a specific time period.
Example: From 1850 To 1900 finds any dataset covering those years
Text Search: Search dataset titles and descriptions for any keyword.
Example: Type "business directory" or "revision lists"
Clickable Tags: Click any tag on a result to instantly filter by that value.
Example: Click "Russian Empire" to see all Russian Empire datasets
New & Updated: Use the Show filter to see only recently added or updated datasets.
Example: Select "New Only" to see what's been added recently
