Welcome to the JewishGen Global Search

Explore millions of records from around the world — including Holocaust and burial records — all in one search.

JewishGen Search Tips:
Comma = OR: Use commas to search for either name.
Example: Schwartz, Weiss (Finds either)
Space = AND: Use a space to search for both names.
Example: Schwartz Weiss (Finds both)
Wildcards (*): Fill in the blanks for bad handwriting (Exact Search).
Example: Sp*l*ng (Finds Spelling, Sperling)
Brackets [ ]: Force a specific starting letter (Phonetic).
Example: [F]isch (Finds Fish, but not Visz)
Synonyms: Exact search includes standard variants.
Example: Jankel = Jakob = Jacob = Jack
Starts With / Contains: Finds root words in the name.
Example: "Gold" finds FeinGold
